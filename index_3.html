🔧 Enable Firebase Realtime Database

Habbit uses Firebase Realtime Database for live sync. It looks like the database isn't reachable for project habbit-7754c.

  1. Firebase Console → Build → Realtime Database
  2. Click Create Database → us-central1 → Test mode
  3. Copy the URL shown (e.g. https://habbit-7754c-default-rtdb.firebaseio.com)
